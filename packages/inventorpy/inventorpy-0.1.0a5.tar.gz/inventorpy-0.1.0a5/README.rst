inventorpy
================================

An inventory management application.

Usage
-----

Installation
------------

::

    pip install inventorpy

Setup development environment
-----------------------------

If you want to work on this project, you need setup your development
environment using the following commands.

::

    $ pip install pipenv
    $ pipenv install --dev
    $ pipenv shell
