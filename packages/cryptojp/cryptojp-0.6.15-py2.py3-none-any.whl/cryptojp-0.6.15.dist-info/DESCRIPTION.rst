cryptojp == Python client for cryptocoin exchanges
==================================================

Description
-----------

-  cryptojp is a python client for crypto coin trade.

HOW TO install
--------------

``pip install cryptojp``

or

``pip install git+https://github.com/airking05/cryptojp``
