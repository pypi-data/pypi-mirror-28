Elastic SDK
===========

|Pypi| |Wheel| |Build Status| |Codecov| |Code Climate|


Python SDK for Elasticsearch


Installation
------------

Install from Pypi.

.. code:: sh

    pip install elastic-sdk


Quick start
-----------

**Client**

.. code:: python

    >>> import elastic_sdk
    >>> client = elastic_sdk.Client(index_name='defocus', hostnames=['elastic:9200'])

**Queries**

.. code:: python

    >>> client.q(user__username='me')


Tests
-----

.. code:: sh

    make test
