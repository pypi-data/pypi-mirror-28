# Thrifty Builder
Example configuration:
```yaml
docker:
  images:
    - name: wtsi-hgi/image-1
      dockerfile: /images/image-1/Dockerfile
      context: /images
    - name: wtsi-hgi/image-2
      dockerfile: /images/image-2/Dockerfile
      # Context assumed to be /images/image-2 
  registries:
    - url: https://docker.io
      username: example
      password: other-example
    
checksum_storage:
  type: consul
  key: /image-checksums
```
