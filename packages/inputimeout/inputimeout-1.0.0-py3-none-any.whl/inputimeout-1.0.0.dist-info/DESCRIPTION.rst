Description
-----------

Multi platform standard input with timeout

Install
-------

.. code:: bash

    $ pip install inputimeout

Usage
-----

.. code:: python

    from inputimeout import inputimeout, TimeoutOccurred
    try:
        something = inputimeout(prompt='>>', timeout=5)
    except TimeoutOccurred:
        something = 'something'
    print(something)
