Install
------------------------------------------------------------------------------

``pytq_crawlib`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install pytq_crawlib

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade pytq_crawlib
